# Talabat API (ASP.NET Core Web API)

## Overview
This project is a RESTful API built with **ASP.NET Core 6** following clean architecture principles.  
It provides endpoints for managing product catalog including **Products, Brands, and Categories**.

## Features
- Get all products
- Get product by ID
- Get all brands
- Get all categories
- Swagger/OpenAPI documentation

## Endpoints
- `GET /api/Products` → Get all products
- `GET /api/Products/{id}` → Get product by ID
- `GET /api/Products/brands` → Get all brands
- `GET /api/Products/categories` → Get all categories
- `GET /api/WeatherForecast` → Demo endpoint

## Technologies
- ASP.NET Core 6
- Entity Framework Core
- SQL Server
- Swagger / OpenAPI
